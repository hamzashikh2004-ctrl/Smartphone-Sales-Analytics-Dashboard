# Smartphone Sales Analytics Dashboard

# Project Overview

This project is an interactive **Power BI dashboard** designed to analyze smartphone sales performance across different brands, regions, customer demographics, and sales channels.

The dashboard enables users to monitor key business metrics, identify sales trends, and compare brand performance through interactive visualizations.
